=====================================================

STEP 1: Copy Files to XAMPP
============================
1. Copy the entire "web_files" folder contents to your XAMPP htdocs directory
   
   FROM: C:\Users\sipar\risk management\web_files\
   TO:   C:\xampp\htdocs\risk_management\
   
   Final structure should be:
   C:\xampp\htdocs\risk_management\
   ├── config\
   │   └── database.php
   ├── auth\
   │   ├── login.php
   │   └── logout.php

STEP 2: Test Database Connection
===============================
1. Make sure XAMPP is running (Apache + MySQL)
2. Open browser and go to: http://localhost/risk_management/test_connection.php
3. You should see "✅ Database connection successful!"
